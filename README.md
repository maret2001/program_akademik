# program_akademik
## Bahasa Indonesia
Program sederhana tentang deskripsi/keterangan akademik yang diinputkan oleh user. Dengan output terdiri dari:
- Nama
- Jenis Kelamin
- Kategori golongan umur
- Asal Kampus
- Status Mahasiswa berdasarkan semester berjalan
- Tahun Masuk
- Tahun Kelulusan
